# Lottery Number Generator

A clean and user-friendly mobile application that generates lucky numbers for various lottery games in Malaysia and beyond.

## 📱 App Overview

This app provides multiple lottery number generators in one place with a consistent, modern, and easy-to-use interface. Users can quickly generate numbers for popular lottery formats including 6D, TOTO Supreme 6/58, and Magnum Life.

## ✨ Features

- **Consistent UI/UX** across all screens
- **Easy Navigation** with bottom navigation bar
- **Multiple Lottery Games** in one app
- **Customizable Generation** (number of sets, manual input option)
- **Reset & Generate** functionality on every screen
- **Responsive and clean number display** (button-style chips)
- **Photo/Banner area** maintained across all screens for branding

## 🎮 Available Programs

### 1. Original Program
The original lottery generator screen (maintained for existing users).

### 2. 6D Permutation
- Generates 6-digit numbers
- Supports manual input or random generation
- Limited to maximum 10 sets per generation

### 3. TOTO Supreme 6/58
- Customizable numbers per ticket (default 6, up to 10)
- Range: 1 to 58
- Generate multiple sets (default 2, max 10)

### 4. Magnum Life
- Customizable numbers per ticket (default 8, up to 10)
- Range: 1 to 32
- Generate multiple sets (default 5, max 10)

### 5. About Page
App information, disclaimer, and developer details.

## 🧭 Navigation
